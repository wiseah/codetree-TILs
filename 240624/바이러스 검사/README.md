# ✏️ 풀이 방식

1. 함수 호출 및 변수할당
2. 팀장이 처리하고 남은 고객 수를 팀원이 검사할 수 있는 고객수로 나눠서 필요한 팀원 수를 구한다.
3. 딱 나누어지지 않는 경우는, 남은 인원들을 팀원 한 명이 추가로 검사해야 한다.
